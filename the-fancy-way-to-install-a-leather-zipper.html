The Fancy Way to Install a Leather Zipper

A long while back we did a full Zipper instillation video:
https://www.youtube.com/watch?v=lT-_b-PX2Kw&t=602s

This tutorial is a continuation on that video, showing you a fancier, more versatile way to install zippers in your leathercraft project. This method allows for zippers to be installed in any different area of a bag, pouch, or even large wallet bu using 4 panels for flexibility. It also can feature a full lining on the back if you're making a lined project! happy leather crafting :)
