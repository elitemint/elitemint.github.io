The song "An Extra Spoon" was inspired by "The Spoon Theory" written by Christine Miserandino. Christine used the analogy of starting each day with a set number of spoons and having to give up a spoon with each daily task completed to explain what it is like to have a limiting chronic illness. If you haven't read it before, I highly recommend it. The process of researching and writing this song was so enlightening for me. I hope you enjoy it. Words are included below:
