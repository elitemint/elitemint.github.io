6. Blender 2.91, a Customised Start-up File - set up your Blender software to have a layout that suits your working preferences. This video illustrates how to prepare and save a custom start-up layout, how to use multiple window views and why you should want to do this.

This series of videos has been put together so information builds from one video to the next. With potters and ceramic artists in mind, a number of computer 3D drawing techniques are explored with clay 3D printing in mind.
