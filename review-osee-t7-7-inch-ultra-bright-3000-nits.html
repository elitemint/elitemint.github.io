【Accurate Video Assisting Tools】 Waveform, false color, focus assist, histogram, auto rotate, anamorphic etc etc. T7 has a full set of image assisting tools to help you nailing framing, exposure, focus and audio. 【Convenient and Clean Operation】 T7’s scene style UI allows user building up to 8 scenes containing different tools for various monitoring situation, after presetting these scenes, all you need to do on set is switching scenes.
