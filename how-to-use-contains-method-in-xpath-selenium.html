XPath expression is an amazing way to locate webelement in selenium webdriver. XPath supports multiple methods. One of such method is contains which is used to locate web elements based on attributes and display text either complete text or partial text/attribute. Basically contains method is very useful when it comes to match the partial value of an attribute or display text. There are two different syntax where we can write contains method.
1. //a[contains(@attribute, 'value')]
2. //a[contains(text(), 'display text')]
