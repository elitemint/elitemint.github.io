I'm entirely fine with the police sometimes making errors. In the roughly 1,000 reports I've made, I've made a couple of errors myself. I think the rider still got an NIP (notice of intended prosecution) so would at least have been worried about that for a while before the admin error came to light. It's not as good a behaviour change agent as getting 3 points, but it should still have an effect on him.
