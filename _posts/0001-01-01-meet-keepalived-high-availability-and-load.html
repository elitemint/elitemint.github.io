This is a protocol that you see on some routers and has been implemented in keepalived. It creates a Virtual IP (or VIP, or floating IP) that acts as a gateway to route traffic to all participating hosts. This VIP that can provide a high availability setup and fail over to another host in the event that one is down. In this video, we'll set up and configure keepalived, we'll test our configuration to make sure it's working, and we'll also talk about some advanced use cases like load balancing.
