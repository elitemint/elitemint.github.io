Exploring the world of Forza Horizon 5 through all seasons and weather conditions.
Huge thanks to Forza for giving me access to the game early! Been slowly gathering footage for this over the last weeks but struggled with deciding when to stop, there is so much to explore, so much weather and time of day combinations, the same location can look so much different a few minutes later and change looks completely another few minutes later. There was so much more I wanted to capture, but I'm already 2 weeks late, so this is where we stop.
And it was a bit too late when I noticed that some footage came out with some stutters, even tho I had no issues running the game maxed out, no clue what was causing it. There can be a laggy recording, a smooth one, and another laggy recording in a row while doing it all in the same location with the same weather conditions in a matter of 3 minutes.
