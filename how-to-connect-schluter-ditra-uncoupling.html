By using Schluter®-DITRA uncoupling and waterproofing membrane to connect to the bathtub surround—you can be sure to achieve a watertight connection that results in a long lasting tile installation. This video walks you through how to connect DITRA to a bathtub surround assembly using Schluter®-KERDI accessories. Check it out for all the details!
